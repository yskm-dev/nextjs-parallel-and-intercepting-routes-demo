# nextjs-parallel-and-intercepting-routes-demo

Next.js App Router の以下を学ぶためのデモプロジェクトです。

- Parallel Routes
- Intercepting Routes

## 開発環境

- Next.js (App Router)
- TypeScript
- React
- npm

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いて確認します。

## よく使うコマンド

```bash
npm run dev
npm run lint
npm run build
```
